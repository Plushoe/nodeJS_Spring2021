VSCode

The IDE I used to write my NodeJS programs is the light-weight and flexible Visual Studio Code. This software has many benefits, one of the big ones being language independence. By installing extensions through the in-app market, VSCode can be fitted for many different tasks - in this case, writing NodeJS!

Furthermore, VSCode comes packed with a terminal, which is critical for NodeJS since compiling and running those programs require command-line use. Lastly, fitting with its flexibility, it's easy to customize features of your VSCode such as different keyboard-shortcuts, or changing the in-app terminal to run bash instead of cmd on Windows machines.
